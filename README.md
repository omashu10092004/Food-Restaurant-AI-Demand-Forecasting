# 🍽️ Food & Restaurant Services - AI Demand Forecasting and Inventory Optimization

## 📌 Project Overview

This project is an AI-powered Restaurant Demand Forecasting and Inventory Optimization System developed to help restaurants predict future customer demand, optimize inventory planning, reduce food waste, and improve business intelligence using Machine Learning and Data Analytics.

The system uses advanced forecasting models, business analytics dashboards, and an interactive AI-powered web application.

✅ Inventory Optimization System
